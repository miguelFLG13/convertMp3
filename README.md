# Convert Mp3 v1.0

#ENG

Script to normalize a lot of mp3 files in a directory. 

   
## Getting Started
This script is easy to use.

Usage:
	python convertMp3.py ./directory/with/mp3

Remember: You need introduce files without strange characters in the name, if not skips the file.

## Examples

```bash
python convertMp3.py ./dir
```

Show help:

```bash
python convertMp3.py help
```

#SPA

Script para normalizar todos los archivos en mp3 contenidos en un directorio.

   
## Comenzar a usarlo
Este script es fácil de usar.

Uso:
	python convertMp3.py ./directorio/con/mp3

Recuerda: Necesitas introducir archivos sin caracteres extraños en el nombre, si no se los saltará.

## Ejemplos

```bash
python convertMp3.py ./dir
```

Mostrar ayuda:

```bash
python convertMp3.py help
```
